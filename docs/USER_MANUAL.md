# User Manual

## Getting Started

### System Overview
The Student Management System is designed to manage student information, academic streams, subjects, and assessment scores.

## Features

### 1. Student Management
- Add, edit, and delete student records
- Track student information including registration number, name, and date of birth
- Assign students to streams

### 2. Stream Management
- Manage academic streams (e.g., Class 10A, Class 10B)
- Organize subjects by stream

### 3. Subject Management
- Create and manage subjects
- Assign subjects to specific streams

### 4. Assessment Scoring
- Record student scores for different assessments (tests, assignments, exams)
- Track scores by subject and student

### 5. Report Generation
- Generate individual student reports with grades
- Generate class reports with statistics
- Generate ranking reports showing student rankings

### 6. Grade Calculation
- Automatic grade calculation based on marks
- Grade scale: A (90-100), B (80-89), C (70-79), D (60-69), E (50-59), F (<50)

## How to Use

### Adding a Student
1. Navigate to Students page
2. Click "Add New Student"
3. Fill in the required information
4. Select the appropriate stream
5. Click "Save"

### Recording Scores
1. Navigate to Scores page
2. Click "Add Score"
3. Select student and subject
4. Enter marks and assessment type
5. Click "Save"

### Generating Reports
1. Navigate to Reports page
2. Select report type (Student, Class, or Ranking)
3. Select the relevant stream or student
4. Click "Generate"
